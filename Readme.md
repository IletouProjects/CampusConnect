# CampusConnect

CampusConnect est un projet web fil rouge réalisé dans le cadre du challenge de **Les Pros de la Tech Cohorte2**.

## Présentation

CampusConnect est une plateforme web pensée pour centraliser les informations importantes de la vie étudiante.

L’objectif du projet est de permettre aux étudiants de retrouver plus facilement :

- les annonces du campus ;
- les ressources pédagogiques ;
- les événements étudiants ;
- les opportunités de stage, de formation ou de concours ;
- les informations utiles publiées par les responsables ou l’administration.

## Fonctionnalités réalisées

Le projet intègre progressivement plusieurs fonctionnalités :

- une page d’accueil responsive ;
- une navigation claire ;
- des sections de présentation du projet ;
- une galerie de contenus filtrable ;
- un générateur d’idées aléatoires ;
- un calculateur de priorité d’annonce ;
- une liste dynamique de tâches avec dates et statuts ;
- un formulaire de contact avec validation ;
- un quiz interactif sur CampusConnect ;
- une FAQ ;
- un footer complet.

## Technologies utilisées

- HTML
- Tailwind CSS
- JavaScript
- Git
- GitHub

## Objectif pédagogique

Ce projet permet de pratiquer les bases du développement web tout en construisant progressivement une interface complète, cohérente et fonctionnelle.

Les notions travaillées incluent :

- structure HTML ;
- mise en page responsive ;
- composants avec Tailwind CSS ;
- manipulation du DOM avec JavaScript ;
- événements utilisateur ;
- validation de formulaire ;
- stockage local avec localStorage ;
- gestion de version avec Git ;
- publication du projet sur GitHub.
