# CampusConnect

CampusConnect est un projet web fil rouge réalisé dans le cadre du challenge **Les Pros de la Tech — Cohorte 2**.

## Présentation du projet

CampusConnect est une plateforme web pensée pour centraliser les informations importantes de la vie étudiante.

Sur un campus, les informations sont souvent dispersées entre plusieurs canaux : groupes WhatsApp, affichages physiques, réseaux sociaux, emails ou messages informels. Cette dispersion peut entraîner des oublis, des retards ou une mauvaise circulation de l’information.

L’objectif de CampusConnect est donc de proposer un espace simple, clair et accessible où les étudiants peuvent retrouver rapidement les informations utiles à leur quotidien.

La plateforme permet notamment de consulter :

- les annonces du campus ;
- les ressources pédagogiques ;
- les événements étudiants ;
- les opportunités de stage, de formation ou de concours ;
- les informations publiées par les responsables, les clubs ou l’administration.

## Objectifs du projet

CampusConnect vise à :

- améliorer l’accès aux informations importantes ;
- réduire la dispersion des messages entre plusieurs canaux ;
- faciliter l’organisation des étudiants ;
- rendre les opportunités plus visibles ;
- proposer une interface moderne, responsive et simple à utiliser ;
- construire progressivement une plateforme complète à travers les exercices du challenge.

## Fonctionnalités réalisées

Le projet intègre progressivement plusieurs fonctionnalités :

- page d’accueil responsive ;
- navigation claire avec menu adapté aux différents écrans ;
- sections de présentation du projet ;
- galerie de contenus filtrable par catégorie ;
- modale de détail pour afficher plus d’informations sur un contenu ;
- générateur d’idées aléatoires ;
- calculateur de priorité d’annonce selon une date ;
- liste dynamique de tâches avec dates, statuts, modification et suppression ;
- sauvegarde des tâches avec `localStorage` ;
- formulaire de contact avec validation en temps réel ;
- quiz interactif sur CampusConnect avec score et timer ;
- FAQ interactive ;
- footer complet avec liens et informations de contact.

Avant d’installer ou de lancer le projet, assurez-vous d’avoir :

- Git installé sur votre ordinateur ;
- Visual Studio Code ou un autre éditeur de code ;
