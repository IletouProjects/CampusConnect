Pourquoi CampusConnect ?

Une solution simple pour mieux organiser les informations du campus

CampusConnect est un projet de mini-site web destiné à aider les étudiants à accéder plus facilement aux informations importantes de leur campus.

Le projet permet de regrouper les annonces, les ressources pédagogiques, les événements étudiants et les opportunités dans un seul espace simple à consulter.

FAQ

Questions fréquentes sur CampusConnect

Retrouvez les réponses aux principales questions sur le fonctionnement et l’objectif de CampusConnect.

Qu’est-ce que CampusConnect ? + -

CampusConnect est une plateforme web pensée pour centraliser les informations importantes de la vie étudiante : annonces, ressources, événements et opportunités.

À qui s’adresse CampusConnect ? + -

La plateforme s’adresse aux étudiants, responsables de classe, clubs étudiants, enseignants et membres de l’administration du campus.

Quels types d’informations seront disponibles ? + -

CampusConnect pourra regrouper les annonces officielles, les supports de cours, les événements étudiants, les opportunités de stage, les formations et les concours.

Pourquoi ce projet est-il utile ? + -

Le projet évite la dispersion des informations entre plusieurs canaux comme les groupes de discussion, les réseaux sociaux, les affichages physiques ou les messages informels.

Comment CampusConnect va-t-il évoluer ? + -

Le projet commence avec une interface simple, puis pourra évoluer avec un espace utilisateur, une gestion des annonces, une base de données, des notifications et des fonctionnalités interactives.
